## ğŸ”§ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Gaurav00780/cross-platform-file-sharing.git
cd cross-platform-file-sharing
```

### 2. Install dependencies

```bash
npm install
```

### 3. Create `.env` file

Inside the root folder:

```
VITE_SUPABASE_URL=your-supabase-project-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
```

### 4. Start development server

```bash
npm run dev
```

## ğŸ”— How It Works (Quick Overview)

1. User selects a file
2. File is uploaded to **Supabase Storage**
3. App generates a public URL or a signed download URL
4. User copies and shares it
5. Receiver opens link and downloads the file
